# 이정수 포트폴리오

MMORPG 시스템/밸런스 기획자 이정수의 경력 포트폴리오입니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 구성

- 대표 프로젝트 6개: 지옥 성채, 길드 협동 보상, 길드 공허 소환체, 장신구 재련, 길드 연구/의뢰, PvP TTK 개선 제안
- 상세 구조: 시스템 소개, 업무 배경, 분석 기준, 내 역할, 설계/검토 방향, 실제 결과, 기대 효과
- 내부 수치: 공개 가능한 범위로 재가공 및 비식별 처리
